# [Valid Anagram - LeetCode](https://leetcode.com/problems/valid-anagram/description/)
## Problem
Given two strings `s` and `t`, return `true` if `t` is an anagram of `s`, and `false` otherwise.
- NOTE: An anagram is a word or phrase formed by rearranging the letters of a different word or phrase, using all the original letters exactly once.
## Approach
- Check length of `s` and `t`, if they aren't equal return False.
- Create 2 hash tables `s_hash` and `t_hash` iterate through both `s` and `t` and add each letter to the hash table with the correct number of instances of that letter (can be done in the same loop because they are the same length).
- Loop through `s` again in order to check if every letter in `s` (key in `s_hash`) has an equivalent key-value pair in `t_hash`.
- If it doesn't, return False.
- After the for loop ends, return True.
## Solution
```python title=valid_anagram_sol.py
class Solution(object):
    def isAnagram(self, s, t):
        """
        :type s: str
        :type t: str
        :rtype: bool
        """
        if len(s) != len(t):
            return False
        s_hash = {}
        t_hash = {}
        for i in range(len(s)):
            s_hash[s[i]]=s_hash.get(s[i],0)+1
            t_hash[t[i]]=t_hash.get(t[i],0)+1
        for i in s:
            if s_hash[i] != t_hash.get(i,0):
                return False
        return True
```
## Complexities
- Time: `O(s+t)`
- Space: `O(s+t)`
- Both `O(n)` technically.
