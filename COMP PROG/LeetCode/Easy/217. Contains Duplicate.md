# [Contains Duplicate - LeetCode](https://leetcode.com/problems/contains-duplicate/description/)

## Problem
Given an integer array `nums`, return `true` if any value appears **at least twice** in the array, and return `false` if every element is distinct.
## Approach
- Want to only loop through the array once as we want O(n).
- Can use a hash table in order to add keys with a value, if the key already exists then there has to be a duplicate.
## Solution
```python title=contains_duplicate_sol.py
class Solution(object):
    def containsDuplicate(self, nums):
        """
        :type nums: List[int]
        :rtype: bool
        """
        temp = {}
        for i in nums:
            if i in temp:
                return True
            temp[i] = 1
        return False
```
## Complexity
- Time: O(n)
- Space: O(n)
