# Course Objectives
-  Describe the role and core responsibilities of an operating system 
- Explain and evaluate key OS abstractions and mechanisms 
- Analyze and compare scheduling, memory management, and storage strategies  
- Implement and extend functionality in an operating system  
- Apply systems-level thinking to reason about performance, reliability, and correctness  
- Write and debug multi-threaded programs
